# LedgerPilot ASP Listing Package

## ASP Name

LedgerPilot

## Category

Finance Copilot

## ASP Type

Agent-to-Agent

## Short Description

LedgerPilot helps users and teams make safer on-chain finance decisions by turning wallet history, stated goals, and risk limits into fixed-scope reports delivered through A2A escrow.

## Services

1. Wallet Risk Triage - $25 starting price
   Reviews approvals, exposure, bridge readiness, concentration risk, and immediate next actions.

2. DeFi Yield Plan - $45 starting price
   Compares pools, estimates net yield, states assumptions, and gives a conservative allocation plan.

3. Treasury Ops Brief - $80 starting price
   Summarizes inflows, outflows, runway, stablecoin mix, controls, and next-week treasury actions.

## Operator Script

1. Open LedgerPilot.
2. Click `Autofill client order`.
3. Show how the selected ASP service updates the escrow quote, ETA, risk posture, and report.
4. Explain that A2A escrow protects both user and provider: the user approves the evidence-backed report before payment release.
5. Copy the listing block for OKX.AI.
